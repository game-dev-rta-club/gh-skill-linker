# Idea

Skill Linker returns skills improved inside one project to their source so
other projects can use those improvements.

The tool does more than distribute skills. It carries lessons learned during
real use into the next installation.

- `skill`: a folder containing instructions and supporting files for an agent
- `local skill`: the copy used and edited inside the current project
- `source`: the distribution location inside a GitHub repository

1. [Goals](pages/goals.md): the outcome the project aims to create
2. [Values](pages/values.md): the criteria used to make tradeoffs
3. [Boundaries](pages/boundaries.md): what the product does and does not cover
